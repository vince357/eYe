# 🖼️ eYe — A Xee³ Successor for Apple Silicon

> A fast, lightweight image viewer for macOS, built natively in Swift/SwiftUI.  
> Born out of the need to replace [Xee³](https://theunarchiver.com/xee), which is no longer maintained and will stop working on Apple Silicon Macs.

---

## ✨ The story behind this project

Xee³ is a beloved macOS image viewer that sadly is no longer maintained. As Apple Silicon Macs progressively drop support for legacy x86 apps, users needed a modern, native alternative that preserves the same lightweight and keyboard-driven experience.

**The idea came from a user. The code was entirely generated by [Claude](https://claude.ai) (Anthropic's AI assistant), in a single conversation.** No manual coding was involved — just a human with a need, and an AI that built the solution. This project is shared as-is in that spirit: open, transparent, and community-improvable.

---

## 📋 Features

- **Full-screen & windowed mode** — toggle with `F` or the button in the HUD
- **Fit-to-window zoom** — image always fully visible on open, no cropping
- **Keyboard zoom** — `+` / `-` to zoom, `0` to reset to fit
- **Folder navigation** — `Page Down` / `Page Up` to go forward/back, with **loop** (wraps around at end/start)
- **Jump to first image** — `Home` key
- **Random image** — `X` key or shuffle button
- **Sort images** by name, creation date, modification date, or file size
- **Include subfolders** — optional recursive navigation toggle
- **Rotation** — rotate 90° left (`R`), as many times as needed
- **Save rotated image** — overwrites the original file (`⌘S`)
- **Reveal in Finder** — `⌘F`
- **Open in default editor** — `⌘E` (opens in Preview, Photoshop, or whatever you've set)
- **Drag & drop** — drop any image onto the window to open it
- **Animated GIFs** — fully supported with native animation playback

### Supported formats

| Category | Formats |
|----------|---------|
| Common | JPG, PNG, GIF (animated), WebP, BMP, ICO |
| Apple | HEIC, HEIF |
| TIFF | TIFF, TIF |
| Vector / Document | SVG, PDF |
| RAW | CR2, CR3, NEF, NRW, ARW, DNG, ORF, RAF, RW2, SRW, PEF, MRW, ERF, and more |
| Other | TGA, EXR, PSD, HDR, PNM, PBM, PGM, PPM |

## 🚀 Getting started

### Requirements
- macOS 13 (Ventura) or later
- Xcode 15 or later ([free on the Mac App Store](https://apps.apple.com/app/xcode/id497799835))

### Build & run

```bash
# 1. Clone the repo
git clone https://github.com/vince357/eYe.git
cd eYe

# 2. Open in Xcode
open eYe.xcodeproj
```

Then in Xcode:
1. Go to the **Signing & Capabilities** tab
2. Select your Apple ID under **Team** (a free account works fine)
3. Press **⌘R** to build and run

### If macOS blocks the app

Go to **System Settings → Privacy & Security** and click **"Open Anyway"**, or run:

```bash
xattr -cr /path/to/eYe.app
```
